# Front Vehicle Brake Lab

This folder is a focused L4/OpenCode/CARLA lab for one risk only:
`lead_vehicle_hard_brake` using the shared action primitive
`front_vehicle_brake_after_trigger`.

It bypasses L1/L2/L3 risk imagination, but keeps the real L4 backend shape:

`l4_task_source.json -> semantic_primitives.json -> l4_task.json -> OpenCode -> generated_risk_scene.scenic -> CARLA -> event_trace.json -> repair feedback`.

The goal is to debug the front-vehicle-brake action primitive and the OpenCode
codegen prompt/template before reinserting it into the full pipeline.

Example:

```bash
python carla_smoke/front_brake_lab/run_front_brake_lab.py \
  --l0-json /path/to/l0/state.json \
  --carla-root "$CARLA_ROOT" \
  --host 127.0.0.1 \
  --port 2001 \
  --output-dir carla_smoke/workdir/front_brake_lab/manual_01 \
  --execute \
  --validate-event-trace
```

Primary outputs:

- `l4_task_source.json`: instantiated L4 task with the concrete action primitive.
- `semantic_primitives.json`: L4 primitive graph sent to OpenCode.
- `opencode_scenario_language_workspace/l4_task.json`: single OpenCode business input.
- `opencode_scenario_language_workspace/generated_risk_scene.scenic`: the single Scenic script generated by OpenCode.
- `selected_front_actor.json`: object selected from L0, if provided.
- `risk_images/event_trace.json`: reconstructed execution trace.
- `l4_feedback_report.md`: execution/semantic validation and repair feedback.
